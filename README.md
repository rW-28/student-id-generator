# student-id-generator
Simple landing page with HTML, CSS, and JS.

Student ID Card Generator
A simple web app built with HTML, CSS, and JavaScript that generates a digital student ID card based on user input.
(**University of Oregon is just chosen for an example(liked the colors)**)

Features
- Input form for name, date of birth, course, and phone number.
- Upload and display of user photo.
- Automatically generates a unique ID number.
- Smooth navigation between pages using localStorage.

Technologies Used
- HTML5  
- CSS3  
- JavaScript (DOM, localStorage, FileReader API)

Live Demo
- [Open the project here](https://rw-28.github.io/student-id-generator/)

What I Learned
- Working with **localStorage** to pass data between pages.  
- Reading and displaying uploaded images via **FileReader**.  
- Designing a simple, responsive layout with CSS.  
- Structuring a small front-end project and publishing it online.
